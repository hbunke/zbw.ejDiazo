zbw.ejDiazo Installation
------------------------

To install zbw.ejDiazo using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``zbw.ejDiazo`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        zbw.ejDiazo
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
